## FAQ

### 1. Is this tool free?

Yes. The audit is completely free to use.

---

### 2. Do I need to create an account?

No. You can use the audit without signing up.

---

### 3. What AI tools are supported?

The tool supports:

- ChatGPT
- Claude
- Cursor
- GitHub Copilot
- Gemini
- OpenAI API
- Anthropic API
- and more

---

### 4. How does the audit work?

You enter:

- your AI tools
- current plans
- monthly spending
- team size
- primary use case

The tool then checks:

- if you are overpaying
- if a cheaper plan fits your usage
- if another tool could give similar value at lower cost

---

### 5. Is my data public?

No. Personal details like email and company name are not shown publicly.
Only the audit results can be shared through a public link.

---

### 6. What if my setup is already good?

The tool will tell you honestly.

If your current setup already makes sense, the audit will say you are spending efficiently instead of forcing unnecessary recommendations.

---

### 7. Why should I trust the recommendations?

The recommendations are based on:

- current pricing data
- team size
- usage type
- plan comparison logic

The goal is to give simple and realistic suggestions instead of random AI-generated advice.

---

### 8. How long does the audit take?

Usually less than 2 minutes.
